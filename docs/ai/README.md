# 智能图像

## 功能介绍
目前腾讯云提供了 `内容审核`、`图片标签`、`文字识别`、`人脸识别`、`人脸融合`、`人脸核身` 等几大功能，本 SDK 暂时开放了 `人脸识别`、`人脸融合`、`人脸核身`功能。

## 准备

在开始使用智能图像服务之前，需要准备如下信息：

- [x] 获取 `SecretID` 和 `SecretKey`

在 [云 API 密钥](https://console.cloud.tencent.com/cam/capi) 获取。
