# 获取实名核身结果信息

[文档](https://cloud.tencent.com/document/api/1007/31331)

## 参数说明

* `service` 值为 `ai`
* `action` 值为 `GetDetectInfo`
* `data` 详情：

| 字段 | 类型 | 必填 | 默认值 | 说明
| --- | --- | --- | --- | ---
| BizToken | string | 是 | | 人脸核身流程的标识，调用DetectAuth接口时生成。
| RuleId | string | 是 | | 用于细分客户使用场景，由腾讯侧在线下对接时分配。
| InfoType | string | 否 | | 指定拉取的结果信息，取值（0：全部；1：文本类；2：身份证正反面；3：视频最佳截图照片；4：视频）。如 134表示拉取文本类、视频最佳截图照片、视频。

## 返回值说明

 字段 | 类型 | 说明
| --- | --- | ---
| code | integer | 返回码，0表示成功
| message | string | 返回信息
| data | object | 返回数据

* `data` 详情

 字段 | 类型 | 说明
| --- | --- | ---
| DetectInfo | string | 详细参考文档
| RequestId | string | 唯一请求 ID，每次请求都会返回。定位问题时需要提供该次请求的 RequestId。
